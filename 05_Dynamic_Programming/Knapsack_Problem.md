# 背包问题速记

- 0-1 背包：每件物品最多取一次。
  - 状态：dp[w] = 容量为 w 的最大价值
  - 转移：dp[w] = max(dp[w], dp[w - weight[i]] + value[i])
  - 遍历顺序：物品外层，容量内层从大到小（避免重复取）
  - 复杂度：O(n·W)，空间 O(W)

- 完全背包（可无限次取用同一物品）：
  - 状态与转移同上
  - 遍历顺序：物品外层，容量内层从小到大（允许多次取）
  - 复杂度：O(n·W)，空间 O(W)

输入格式（与示例代码一致）：
- 第一行：n W
- 接下来 n 行：wi vi

备注：
- 若你指“恰好装满”的变体，可把 dp 初始化为 -INF，dp[0]=0，并在输出时取 dp[W]。
